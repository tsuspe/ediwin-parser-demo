# Samples — EDIWIN Parser Demo

Este directorio contiene ejemplos ficticios para probar la aplicación:

- `eurofiel/` → PDFs de ejemplo con estructura similar a Eurofiel.
- `eci/` → PDFs de ejemplo con estructura similar a El Corte Inglés (EDIWIN).
- `eci_txt/` → Ficheros TXT de ejemplo El Corte Ingles (LINPED, CABPED, etc.) con datos inventados.
- `eurofiel_txt/` → Ficheros TXT de ejemplo Eurofiel (LINPED, CABPED, etc.) con datos inventados.

Todos los datos de este directorio son ficticios y se usan únicamente con fines demostrativos.
